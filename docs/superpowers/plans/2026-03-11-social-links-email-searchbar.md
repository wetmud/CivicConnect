# Social Links, Modal Email & Search Bar Polish Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Auto-populate rep social links via Wikidata, display clickable email in the profile modal, and give the search bar a pill shape with shadow.

**Architecture:** All changes are in `index.html` (single-file vanilla JS/CSS project). Wikidata fetches run in parallel with existing Wikipedia bio fetch on modal open. Social merge favours Represent API data, fills gaps with Wikidata. Email display is a click-to-copy element in the modal left column. Search bar changes are CSS-only.

**Tech Stack:** Vanilla JS, CSS custom properties, Wikidata REST API (no key), Clipboard API

**Security note on innerHTML:** All social link HTML is generated by the existing `buildSocialLinks()` helper, which sanitizes every value through `escAttr()` before inserting into HTML attribute positions. This matches the existing codebase pattern (see line ~2382). No raw user or API data is inserted unsanitized.

function rpModalCopyEmail() {
  const el = document.getElementById('rpModal-email-display');
  const email = currentRepForProfile && currentRepForProfile.email;
  if (!email) return;
  navigator.clipboard.writeText(email).then(() => {
    el.textContent = 'Copied!';
    el.classList.add('copied');
    setTimeout(() => {
      el.textContent = email;
      el.classList.remove('copied');
    }, 1500);
  });
}

- [ ] **Step 2: Add `fetchWikidataSocials()` async function**

Add after `fetchWikiBio()`. All Wikidata values flow through `buildSocialLinks()` -> `escAttr()` before any HTML insertion.

```js
async function fetchWikidataSocials(rep) {
  const name = rep.name;
  if (wikidataCache[name] !== undefined) {
    renderModalSocials(rep, wikidataCache[name]);
    return;
  }
  try {
    const searchUrl = 'https://www.wikidata.org/w/api.php?action=wbsearchentities&search='
      + encodeURIComponent(name)
      + '&language=en&type=item&limit=3&format=json&origin=*';
    const searchRes = await fetch(searchUrl);
    const searchData = await searchRes.json();
    const results = searchData.search || [];
    if (!results.length) {
      wikidataCache[name] = {};
      renderModalSocials(rep, {});
      return;
    }

    const entityId = results[0].id;
    const propsUrl = 'https://www.wikidata.org/w/api.php?action=wbgetentities&ids='
      + entityId
      + '&props=claims&format=json&origin=*';
    const propsRes = await fetch(propsUrl);
    const propsData = await propsRes.json();
    const claims = (propsData.entities[entityId] && propsData.entities[entityId].claims) || {};

    // Map Wikidata property IDs to social keys used by buildSocialLinks()
    const WIKIDATA_PROPS = [
      { prop: 'P2002', key: 'twitter',   base: 'https://twitter.com/' },
      { prop: 'P2003', key: 'instagram', base: 'https://instagram.com/' },
      { prop: 'P2013', key: 'facebook',  base: 'https://facebook.com/' },
      { prop: 'P2397', key: 'youtube',   base: 'https://youtube.com/channel/' },
      { prop: 'P6634', key: 'linkedin',  base: 'https://linkedin.com/in/' },
    ];

    const extra = {};
    for (const { prop, key, base } of WIKIDATA_PROPS) {
      const claim = claims[prop] && claims[prop][0]
        && claims[prop][0].mainsnak
        && claims[prop][0].mainsnak.datavalue
        && claims[prop][0].mainsnak.datavalue.value;
      if (claim && typeof claim === 'string') {
        extra[key] = base + claim;
      }
    }

    wikidataCache[name] = extra;
    renderModalSocials(rep, extra);
  } catch (e) {
    wikidataCache[name] = {};
    renderModalSocials(rep, {});
  }
}
```

- [ ] **Step 3: Add `renderModalSocials()` helper**

Add directly after `fetchWikidataSocials()`:

```js
function renderModalSocials(rep, wikidataExtra) {
  // Represent API data takes priority; Wikidata fills only empty keys
  const base = rep.extra || {};
  const merged = Object.assign({}, wikidataExtra, base);
  const mergedRep = Object.assign({}, rep, { extra: merged });
  // buildSocialLinks() sanitizes all values via escAttr() internally
  const socialHtml = buildSocialLinks(mergedRep)
    .replace('<div class="rep-social">', '')
    .replace('</div>', '');
  document.getElementById('rpModal-social').innerHTML = socialHtml;
}
```
